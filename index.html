    O material

    O filamento 3d é um material fabricado especialmente para alimentar impressoras 3D que utilizam o método FDM.

    Os filamentos estão para as impressoras 3D como os cartuchos de tinta estão para as impressoras convencionais, ou seja, são os suprimentos que permitem a impressão dos objetos
